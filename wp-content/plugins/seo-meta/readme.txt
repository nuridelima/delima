SEO optimization by automatically adding meta OG and Twitter Cards tags.

== Description ==

Inject automatically, to all your articles, Open Graph meta tags and Twitter Cards:

* og:type
* og:title
* og:description
* og:url
* og:site_name
* og:image
* twitter:card
* twitter:domain
* twitter:image:src

== Installation ==

= WordPress Admin Method =
1. Go to you administration area in WordPress `Plugins > Add`
2. Look for `SEO Meta` (use search form)
3. Click on Install and activate the plugin

= FTP Method =
1. Upload the complete `seo-meta` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
